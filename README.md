# DT208G - Laboration 4 - Elin Ronda

Detta är en angular-applikation som jag har skapat för laboration 4. Applikationen hämtar data från en extern JSON-fil och visar kurserna i en tabell. Man kan sortera kurskod, kursnamn och progression i stigande ordning och man kan även filtrera genom att skriva in bokstäver som matchar kurskod eller kursnamn.

## Funktionalitet
- Häma kursdata med HttpClient
- Använder en service för anrop
- Visar kurskod, kursnamn, progression och kursplan (klickbar länk)
- Filtrera kurskod och kursnamn
- Sortera kurskod, kursnamn och progression
- Tabellen uppdateras direkt utan att den behöver laddas om

### Tekniker

- HttpClient
- Signals
- Computed
- HTML
